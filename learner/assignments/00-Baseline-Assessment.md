# Assignment 00: Baseline Assessment

## Goal

Collect enough evidence to estimate your current English writing level and build your first roadmap.

## What To Do

1. Answer the interview questions in [learner/submissions/00-Initial-Interview-Answers.md](../submissions/00-Initial-Interview-Answers.md).
2. Complete the writing diagnostic in [learner/submissions/00-Baseline-Writing-Diagnostic-Response.md](../submissions/00-Baseline-Writing-Diagnostic-Response.md).

## Rules

- Write honestly.
- Do not use AI to draft or rewrite the diagnostic.
- A dictionary is allowed if you mention it.

## Deliverables

- one completed interview file
- one completed diagnostic file

## When Finished

Tell Codex: `Assess me using the intake files in learner/submissions/`.
