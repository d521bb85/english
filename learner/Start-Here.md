# Start Here

This is your learner dashboard.

## Your First Job

Complete the baseline assessment before we start normal training.

1. Answer the interview in [learner/submissions/00-Initial-Interview-Answers.md](submissions/00-Initial-Interview-Answers.md).
2. Complete the diagnostic in [learner/submissions/00-Baseline-Writing-Diagnostic-Response.md](submissions/00-Baseline-Writing-Diagnostic-Response.md).
3. When both are done, tell Codex: `Assess me using the intake files in learner/submissions/`.

## After The Assessment

Once your assessment is complete, the normal workflow is simple:

1. Say `Let's work`.
2. Do the assignment in `learner/submissions/`.
3. Ask Codex to review it.
4. Repeat.
