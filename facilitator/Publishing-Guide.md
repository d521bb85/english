# Publishing Guide

This framework is designed so the reusable system is separate from personal learner history.

## Generic Files Safe To Publish

- `README.md`
- `AGENTS.md`
- everything in `facilitator/`
- everything in `templates/`
- the structural parts of `learner/` after removing personal answers and feedback

## Personal Files To Review Before Publishing

- `learner/submissions/`
- `learner/feedback/`
- `progress/Learner-Profile.md`
- `progress/Roadmap.md`
- `progress/Mastery-Tracker.md`
- `progress/Session-Log.md`

## How To Turn This Repo Into A Public Template

1. Remove personal responses, teacher feedback, and private goals.
2. Keep the prompt files and blank templates.
3. Reset the progress files to placeholder values.
4. Add an example walkthrough if you want the template to be self-explanatory.

## Recommended Public Structure

For a reusable public version, keep:

- the coaching protocol
- the rubric
- the command vocabulary
- blank intake and assignment templates
- an empty learner workspace

For privacy, remove or anonymize:

- real learner writing
- detailed personal history
- schedules, deadlines, or sensitive motivations
